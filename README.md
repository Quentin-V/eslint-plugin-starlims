# Starlims ESLint Plugin
### This plugin provides custom rules for Starlims.

## List of customizations
- Do not create parsing errors for #import statements
- Allow usage of Starlims global variables such as:
    - 'form'
    - 'lims'
    - 'Shell'
    - 'Starlims'
